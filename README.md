# Wing Browser

**Internet mu Kinyarwanda** - Rwanda's first indigenous web browser

## Overview

Wing Browser is a modern web browser designed and developed in Rwanda, fully localized in Kinyarwanda, and optimized for the digital needs of Rwandan users. It provides a fast, secure, and user-friendly way for citizens to explore the internet while promoting the Rwandan language, culture, and digital independence.

## Vision

To make the internet accessible, understandable, and empowering for all Rwandans through a web browser that speaks their language and reflects their values.

## Features

-**Full Kinyarwanda Interface** - Entire browser in Kinyarwanda
-**Localized Search** - Prioritizes Rwandan and African content
-**Kinyarwanda Spellchecker** - Built-in language tools
-**Voice & Accessibility** - Text-to-speech and voice input
-**Privacy & Safety** - Built-in tracker blocking and security
-**Rwandan Content Hub** - Quick access to local resources
-**Cross-Platform** - Windows, Linux, macOS support

## Development

### Prerequisites

- Node.js 18+ and npm
- Git

### Setup

\`\`\`bash
# Install dependencies
npm install

# Run in development mode
npm run dev:electron

# Build for production
npm run build

# Platform-specific builds
npm run build:win    # Windows
npm run build:linux  # Linux
npm run build:mac    # macOS

## Technology Stack

- **Framework**: Electron 28+ with Chromium
- **UI**: React 18+ with TypeScript
- **State**: Redux Toolkit
- **Build**: Vite
- **Language**: TypeScript
